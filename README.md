# IFFCO Fertilizer Demand Forecasting System

A Flask-based Data Analytics and Machine Learning project developed during the IFFCO Internship.

## Features

- Fertilizer Sales Dashboard
- Data Upload
- Data Analysis
- Demand Forecasting
- Interactive Charts
- CSV Export

## Technologies

- Flask
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- OpenPyXL

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
