# 🌱 IFFCO Fertilizer Demand Forecasting & Sales Analytics System

A full-stack web application developed during my Data Analytics Internship at IFFCO. The system helps monitor fertilizer sales, manage employees, analyze sales performance, and predict future fertilizer demand using Machine Learning.

---

## 📌 Project Overview

The IFFCO Fertilizer Demand Forecasting & Sales Analytics System is designed to help managers and employees efficiently manage fertilizer sales operations. The application provides real-time dashboards, employee management, sales tracking, historical analytics, and AI-based demand forecasting.

This project combines Data Analytics, Machine Learning, SQL, and Flask to provide actionable business insights.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/adarsh-kumar0623/IFFCO.git
```

Go to project folder

```bash
cd IFFCO
```

Install dependencies

```bash
pip install -r requirements.txt
```

Initialize Database

```bash
python database/init_db.py
```

Insert Master Data

```bash
python database/seed_data.py
```

Insert Sales Dataset

```bash
python database/seed_sales.py
```

Run Application

```bash
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
