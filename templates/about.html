IFFCO Fertilizer Demand Forecasting System

A Smart Data Analytics & Machine Learning based Web Application developed to analyze fertilizer sales and predict future demand using historical sales data.

Project Objective

The objective of this project is to help IFFCO managers monitor fertilizer sales across different states and districts, analyze business performance and predict future fertilizer demand using Machine Learning.

  • Sales Analytics Dashboard
  • Employee Management
  • Demand Forecasting
  • Interactive Reports
  • Data Driven Decision Making

Technology Stack

Frontend HTML, CSS, Bootstrap 5, JavaScript
Backend Python Flask
Database SQLite
Machine Learning Scikit-Learn (Linear Regression)
Visualization Chart.js

Key Features

  • Manager Dashboard
  • Employee Dashboard
  • Sales Entry
  • Sales History
  • Employee CRUD
  • Live Charts
  • Analytics Dashboard
  • CSV Report Export
  • Search & Filters
  • SweetAlert Integration
  • Demand Forecast
  • Responsive UI
  • Authentication
  • SQLite Database
  • Professional Dashboard

About IFFCO

Indian Farmers Fertiliser Cooperative Limited (IFFCO) is one of the world's largest fertilizer cooperative organizations. It provides high-quality fertilizers and agricultural solutions to millions of farmers across India.
